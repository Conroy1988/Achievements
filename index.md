## How claims are evaluated

This project distinguishes official GitHub documentation from reproducible tests, dated observations, and community reports. Where GitHub does not publish a rule or threshold, the encyclopedia states that limitation directly.

## Help improve the reference

Useful contributions include corrections, current screenshots, reproducible tests, source updates, accessibility work, and clearer explanations. Artificial engagement and empty activity are not accepted as evidence.
