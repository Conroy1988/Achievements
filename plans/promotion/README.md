# Canonical promotion plans

This directory contains generated or maintainer-approved plans for applying accepted mission-evidence promotion proposals.

A plan is an impact preview and implementation checklist. It does not modify canonical claims, evidence, contradictions, thresholds, missions, release gates, tags, or releases.

Only accepted review-queue entries with a non-`none` promotion target may produce a plan. Every canonical change must be implemented through a separate maintainer-reviewed pull request and isolated commit so it can be reverted cleanly.

Do not place credentials, private repository material, private communications, financial information, or unnecessary personal identifiers in a plan.
