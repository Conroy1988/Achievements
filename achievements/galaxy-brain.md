## Galaxy Brain

**Status:** Earnable  
**Tiered:** Yes  
**Evidence level:** Trigger and thresholds are community documented; GitHub does not publish a complete tier table.

## Trigger

Galaxy Brain recognises answers in GitHub Discussions that are marked as accepted answers.

## Community-reported tiers

| Level | Accepted answers |
|---|---:|
| Base | 2 |
| Bronze | 8 |
| Silver | 16 |
| Gold | 32 |

Treat these values as community-verified rather than guaranteed platform contracts.

## What qualifies

- The response must be posted as an answer in a GitHub Discussion.
- The discussion should use a category that supports answers.
- The answer must be marked as the accepted answer.
- The activity must remain attributable to your account.

## Quality standard

Useful answers should solve the question, explain the reasoning, and avoid copied or low-effort responses. Do not coordinate artificial questions and accepted answers solely to manufacture achievement progress.

## Important distinction

GitHub added a separate verified-answer state for Discussions in 2025. Galaxy Brain has historically been associated with accepted answers; a verified answer is a stronger moderation signal but should not be assumed to be a separate achievement requirement without evidence.

## Troubleshooting

Check that the discussion category supports answers and that the response visibly carries the accepted-answer state. Achievement updates may be delayed.
