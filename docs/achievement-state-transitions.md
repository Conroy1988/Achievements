# Achievement state transitions

Use explicit states when tracking achievement guidance:

- **Unverified** — claim recorded without adequate evidence.
- **Observed** — behaviour seen but not independently reproduced.
- **Reproduced** — qualifying behaviour repeated under documented conditions.
- **Confirmed** — supported by authoritative documentation or strong repeated evidence.
- **Disputed** — credible evidence conflicts with the current claim.
- **Retired** — no longer available to newly qualify.
- **Superseded** — replaced by newer guidance.

Every transition should include a date, reason, and evidence link.
