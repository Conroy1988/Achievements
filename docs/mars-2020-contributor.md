# Mars 2020 Contributor

## Summary

Mars 2020 Contributor is a retired GitHub profile achievement recognising qualifying contributions to open-source repositories used by NASA's Mars 2020 mission. It cannot be newly earned through current activity.

## Historical trigger

The achievement was associated with contributions to repositories identified by GitHub as dependencies or project components used in the Mars 2020 mission.

## Progression or tiers

This achievement is not known to have tiers.

## Eligibility conditions

Eligibility depended on GitHub's historical repository and contributor matching. A repository's connection to Mars 2020 did not necessarily mean that every contributor, fork, or later contribution qualified.

## Award timing

The achievement was awarded after GitHub processed the Mars 2020 dependency and contribution data. It is not an ongoing award.

## Verification

Confirm the badge on the profile achievements section, then compare dated contribution history with the repositories identified in contemporaneous Mars 2020 materials. Avoid publishing private repository data or personal account details.

## Evidence status

- The Mars 2020 mission's use of open-source software is **Official**.
- The existence and retired status of the GitHub achievement are **Confirmed**.
- Exact repository lists, contribution cut-offs, and attribution edge cases may be **Observed** or **Unknown** where GitHub did not publish complete rules.

## Known limitations and edge cases

- The badge cannot be earned through new contributions.
- Contributions made after the historical eligibility window do not qualify retroactively.
- Renamed repositories, transferred ownership, rewritten history, and changed commit-email attribution can complicate retrospective checks.

## Troubleshooting

1. Confirm that profile achievements are visible.
2. Verify that the contribution predates the historical award processing.
3. Check commit attribution to the relevant GitHub account.
4. Distinguish direct contribution from activity only in a fork or unrelated derivative.
5. Record unresolved claims as unknown rather than inferring eligibility.

## Responsible participation

Do not create misleading historical claims or alter repository records to simulate qualification. This page documents a completed historical programme.

## History

GitHub introduced the achievement following the Mars 2020 mission and later retired it from active eligibility.
