# Mars 2020 Contributor

## Summary

Mars 2020 Contributor is a retired GitHub profile achievement recognising qualifying commits to specific open-source versions used by NASA's Mars 2020 Helicopter Mission. It cannot be newly earned.

## Historical trigger

GitHub's current profile reference states that the achievement was given to people who authored a commit for a listed qualifying repository. Archived GitHub documentation adds that the commit had to be present in the listed tag and could be authored or co-authored by the account.

## Progression or tiers

This achievement is not documented as tiered.

## Eligibility conditions

- The repository, version, and tag had to appear in GitHub's published qualifying list.
- The authored or co-authored commit had to be present in that tag's history.
- The commit email had to be verified and associated with the account when GitHub determined eligibility.
- Later changes to verified email addresses did not alter the historical badge decision.

## Verification

1. Use GitHub's current qualifying repository, version, and tag list.
2. Inspect the relevant tag's commit history.
3. Confirm author or co-author attribution to the account using public evidence.
4. Treat account-email association at historical processing time as unavailable unless safely documented.

## Evidence status

- The qualifying list and broad authorship rule are **Official** in maintained GitHub documentation.
- Tag, co-author, and historical verified-email conditions are **Official** in archived GitHub documentation.
- Retrospective verification may remain incomplete where historical account-association evidence is unavailable.

## Known limitations and edge cases

- The event has ended and cannot be earned through new contributions.
- A contribution to another version or tag does not satisfy the published historical rule.
- Renames, transfers, rewritten history, and deleted accounts can complicate retrospective checks.
- Public commit attribution does not always prove the email was associated with the account at GitHub's historical processing time.

## Responsible participation

Do not create misleading historical claims or alter repository records to simulate qualification. This page documents a completed event.

## History

GitHub and JPL published the badge in 2021 to recognise contributors to the specific open-source versions used by Ingenuity.
