# Accessibility Guidance

Documentation should remain usable without relying on badge colour or imagery alone.

- Include achievement and tier names as text beside images.
- Add meaningful alt text to screenshots and badge graphics.
- Use descriptive link text rather than bare URLs or “click here”.
- Keep heading levels sequential and tables readable on narrow screens.
- Do not use emoji as the only indicator of status.
- Explain bronze, silver, and gold tiers in text.

Accessible documentation improves the project for screen-reader users, mobile users, and readers with colour-vision differences.
