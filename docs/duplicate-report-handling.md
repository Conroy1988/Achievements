# Duplicate report handling

When multiple reports describe the same achievement behaviour:

1. Link the reports to a single primary record.
2. Preserve genuinely different environments or outcomes.
3. Avoid counting repeated copies as independent confirmation.
4. Summarise common evidence and material differences.
5. Close duplicates only after retaining useful context.
6. Reopen review when a duplicate introduces new reproducible evidence.

Deduplication should reduce noise without discarding corroboration.
