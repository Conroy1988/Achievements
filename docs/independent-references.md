# Independent References

Undocumented achievement thresholds should not rely on a single community page.

For each numerical claim:

- compare at least two independently maintained references;
- check whether one source copied the other;
- prefer recently maintained sources with visible revision history;
- note material disagreement instead of averaging values;
- treat agreement as corroboration, not official confirmation;
- retain a dated reproduction record when practical.

A source list is strongest when readers can trace both the evidence and the uncertainty behind the conclusion.
