# Pull Shark

Pull Shark recognises pull requests authored by a user and merged into a repository.

## Known progression

- Base achievement: 2 merged pull requests
- Bronze tier: 16 merged pull requests
- Silver tier: 128 merged pull requests
- Gold tier: 1,024 merged pull requests

## Verification notes

Only merged pull requests count. Closed, unmerged pull requests do not contribute. Processing may not be immediate, so profile badges can appear after a delay.

## Responsible route

Use focused pull requests that each deliver an independently reviewable improvement. Avoid empty commits, duplicate changes, or deliberately noisy repository history.
