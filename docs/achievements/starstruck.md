# Starstruck

Starstruck recognises a repository that receives sustained community interest through GitHub stars.

## Known threshold

The base achievement is generally associated with 16 stars on a repository owned by the user.

## Legitimate route

Build something useful, document it clearly, publish examples, and share it with the relevant community. Stars should come from real users who independently value the project.

Buying stars, coordinating fake accounts, or exchanging stars undermines the signal and should be avoided.
