# Pair Extraordinaire

Pair Extraordinaire recognises co-authored commits that are incorporated through a merged pull request.

## Valid attribution

Add a `Co-authored-by:` trailer using the collaborator's verified GitHub email address. Both contributors must have genuinely participated in the change.

## Workflow

1. Collaborate on a substantive change.
2. Add the co-author trailer to the commit message.
3. Push the commit on a branch.
4. Merge the branch through a pull request.

Do not invent collaborators or use another person's identity without their involvement.
