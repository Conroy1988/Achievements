# Public Sponsor

Public Sponsor recognises support provided through GitHub Sponsors.

## Route

Sponsor an eligible open-source developer or organisation through GitHub Sponsors and keep the sponsorship public where required for profile recognition.

## Considerations

Choose a maintainer or project you genuinely rely on. Review the recurring amount, billing currency, privacy setting, and cancellation terms before confirming.

This achievement cannot be generated through repository commits or pull requests because it depends on a completed sponsorship transaction.
