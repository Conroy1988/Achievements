## Threshold boundary programme

A high-tier badge far above a proposed threshold is weak boundary evidence. This programme records the exact below/at/above observations still required.

| Programme | Achievement | Proposed boundaries | Status |
|---|---|---|---|
| `BND-001` | pull-shark | 2, 16, 128, 1024 qualifying merged pull requests | blocked-missing-attributed-counts |
| `BND-002` | pair-extraordinaire | 1, 10, 24, 48 qualifying merged pull requests | blocked-missing-coauthor-counts |
| `BND-003` | galaxy-brain | 2, 8, 16, 32 accepted answers | blocked-missing-accepted-answer-counts |
| `BND-004` | starstruck | 16, 128, 512, 4096 repository stars | bracketed-not-confirmed |
| `BND-005` | quickdraw | 300 seconds after creation | one-sided-observation |

## Required next evidence

### BND-001 — pull-shark

Capture independently attributed exact counts immediately below and at each visible tier transition.

### BND-002 — pair-extraordinaire

Capture final-history co-author attribution and visible tier state around each merge-method boundary.

### BND-003 — galaxy-brain

Capture exact accepted-answer totals and public tier states around each proposed transition.

### BND-004 — starstruck

Obtain exact unrounded star totals immediately below and above the x4 transition with ownership and timing controls.

Current public bracket: **2.1k stars with x3** to **5.1k stars with x4**. The reported boundary is **4096**, but rounded counts do not confirm it.

### BND-005 — quickdraw

Run independent positive and negative controls immediately around the proposed five-minute boundary.
