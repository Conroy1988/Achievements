# Award delay troubleshooting

Use this checklist when qualifying activity has completed but an achievement has not appeared.

1. Confirm the pull request or other qualifying event completed successfully.
2. Confirm the activity is attributed to the expected GitHub account.
3. Check that the repository and relevant activity remain accessible.
4. Allow for profile processing and cache delays before repeating activity.
5. Record the event URL, completion time, and first verification time.
6. Avoid creating duplicate activity solely because the profile has not refreshed.

A delayed badge is not evidence that the qualifying event failed.
