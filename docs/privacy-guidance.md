# Privacy Guidance

Achievement verification often involves profile screenshots, repository links, and timestamps. Contributors must avoid exposing information that is unnecessary to reproduce a result.

- Redact email addresses, access tokens, billing details, private repository names, and unrelated notifications.
- Do not publish screenshots of private activity without permission from everyone identifiable in them.
- Prefer links to public GitHub events over copied personal data.
- Do not reveal another contributor's private email in a co-author trailer.
- Remove sensitive metadata before uploading images or exported files.

Evidence should establish the qualifying event while disclosing the minimum amount of personal information required.
