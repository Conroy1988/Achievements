# Achievement Tier Thresholds

Several GitHub achievements have progressive tiers. Thresholds are community-observed because GitHub does not publish a complete formal specification.

| Achievement | Base | Bronze | Silver | Gold |
|---|---:|---:|---:|---:|
| Pull Shark | 2 | 16 | 128 | 1,024 |
| Pair Extraordinaire | 1 | 10 | 24 | 48 |
| Galaxy Brain | 2 | 8 | 16 | 32 |
| Starstruck | 16 | 128 | 512 | 4,096 |

Treat these values as operational guidance rather than a contractual API. GitHub may alter badge logic or processing behaviour.
