# Achievement Matrix

| Achievement | Practical route |
|---|---|
| Quickdraw | Close an issue or pull request shortly after opening it |
| Pull Shark | Open pull requests that are merged |
| YOLO | Merge a pull request without a review |
| Pair Extraordinaire | Co-author commits included in merged pull requests |
| Starstruck | Build a repository that receives enough genuine stars |
| Galaxy Brain | Have helpful Discussion answers accepted |
| Public Sponsor | Sponsor eligible open-source work through GitHub Sponsors |
