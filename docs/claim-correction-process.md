# Claim correction process

When published guidance is wrong or misleading:

1. Identify the exact statement affected.
2. Preserve the prior wording in history rather than concealing it.
3. Link the evidence that supports the correction.
4. Update related guides, tables, and examples.
5. State whether the correction changes qualification advice.
6. Record the correction date and reviewer.

Corrections should be prompt, transparent, and proportionate to the impact of the error.
