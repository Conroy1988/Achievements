# Glossary

- **Achievement** — a badge displayed on a GitHub profile after qualifying activity.
- **Tier** — a progression level such as default, bronze, silver, or gold.
- **Qualifying event** — the action GitHub evaluates for an achievement.
- **Community-observed threshold** — a numerical requirement repeatedly reported by users but not formally guaranteed by GitHub.
- **Reproduction** — a controlled test intended to confirm a reported behaviour.
- **Accepted answer** — a response marked as the answer in a GitHub Discussion.
- **Co-authored commit** — a commit containing valid attribution for more than one contributor.
- **Merged pull request** — a pull request whose changes have been integrated into its base branch.
