# Frequently Asked Questions

## Are GitHub achievement thresholds officially documented?

Some triggering behaviours are visible in GitHub's interface, but many numerical tier thresholds are community observed. This project labels that distinction explicitly.

## Does a badge appear immediately?

Not always. Profile processing can lag behind the qualifying event, so record the event and re-check later before declaring a test unsuccessful.

## Do private repositories count?

Behaviour may vary by achievement and may change. Tests should always record repository visibility rather than assuming equivalence.

## Can achievements be earned with fake accounts or spam?

This project does not recommend fabricated collaboration, disposable accounts, deceptive sponsorship, or meaningless repository activity.

## Why keep a reproduction record?

A dated record makes it possible to distinguish current behaviour from old community folklore.
