# Deprecation Policy

When an achievement or earning route is no longer available, preserve its historical documentation but mark it clearly.

- Change the status to retired or unavailable.
- Record the date the change was confirmed.
- Preserve the former qualifying condition and thresholds as historical information.
- Explain whether existing holders retain the badge.
- Remove instructions that imply new users can still earn it.
- Link to official retirement evidence where available.

If retirement is only suspected, label the page needs verification instead of presenting the conclusion as settled.
