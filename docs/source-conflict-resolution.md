# Source conflict resolution

When achievement sources disagree:

1. Confirm that each source refers to the same achievement and time period.
2. Prefer current official GitHub documentation over undated summaries.
3. Check whether platform behaviour changed after an older observation.
4. Reproduce the disputed condition where practical.
5. Record both credible interpretations while uncertainty remains.
6. Explain why one conclusion is preferred.
7. Date the decision and schedule re-review when needed.

Do not discard contrary evidence without documenting the reason.
