# Retired and Historical Achievements

Some GitHub profile achievements are tied to historical events or programmes and cannot be newly earned.

## Arctic Code Vault Contributor

Awarded for contributions included in GitHub's 2020 Arctic Code Vault snapshot. The qualifying snapshot is complete.

## Mars 2020 Contributor

Connected to repositories used by the Mars 2020 mission. It is historical rather than an ongoing achievement route.

## Reference policy

Historical achievements are documented for completeness but clearly marked as unavailable so users do not waste effort attempting obsolete workflows.
