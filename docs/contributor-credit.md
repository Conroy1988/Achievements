# Contributor Credit

Credit contributors according to the work they actually performed.

- Use GitHub authorship and co-author trailers only for genuine contributions.
- Mention research, testing, documentation, design, and maintenance contributions where appropriate.
- Obtain permission before publishing personal details not already public on GitHub.
- Do not add fabricated co-authors to pursue an achievement.
- Preserve attribution when adapting externally licensed material.

Accurate credit protects project integrity and makes collaboration history useful to future maintainers.
