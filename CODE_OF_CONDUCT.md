# Contributor Code of Conduct

## Our standards

This project welcomes evidence-led, respectful contributions from people of every background and experience level.

Expected conduct includes:

- discussing claims, evidence, and corrections professionally;
- distinguishing verified facts from community observations;
- respecting privacy, attribution, and repository rules;
- accepting constructive review and correcting mistakes transparently;
- keeping contributions useful, focused, and reproducible.

Unacceptable conduct includes harassment, discrimination, threats, deliberate misinformation, impersonation, disclosure of private information, spam, fabricated evidence, fake engagement, and attempts to manipulate GitHub activity metrics.

## Scope

This policy applies to repository issues, pull requests, Discussions, project documentation, and other spaces representing the project.

## Enforcement

Project maintainers may edit, hide, reject, lock, or remove content that breaches these standards. Repeated or serious violations may result in blocking from project spaces.

## Reporting

For ordinary moderation concerns, open a repository issue without including sensitive personal information. For security or private-data concerns, follow `SECURITY.md` instead of posting details publicly.

## Attribution

This policy is adapted from Contributor Covenant principles and tailored to the evidence and privacy requirements of this project.
