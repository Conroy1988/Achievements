# Mission packet reviews

Each accepted mission packet may have one final review record in this directory.

- Copy `review.template.json` to `MRV-YYYY-NNN.json`.
- Reference the exact packet under `triage/mission-drafts/`.
- Complete every checklist field and disclose reviewer conflicts.
- Use a promotion proposal only when the mission publishes that target and the applicable adjudication rule is satisfied.
- Keep `canonical_mutation` set to `false`.

A review record classifies evidence. It never edits canonical claims, contradictions, thresholds, mission status, or release state automatically.
