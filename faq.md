# GitHub achievements FAQ

## Why has my achievement not appeared yet?

GitHub profile achievements are not always displayed immediately. Processing and caching delays can occur after qualifying activity. Confirm the event first, record its timestamp, and allow time before attempting anything again.

## Does closing a pull request count for Pull Shark?

No. Pull Shark is associated with pull requests that are merged, not merely closed.

## Are achievement tier thresholds official?

Some achievement behaviour is documented or directly observable, but GitHub does not publish a complete official tier table. Thresholds presented by this project are labelled according to their evidence strength.

## Does private repository activity count?

Private activity may qualify for some platform features, but it is difficult for independent researchers to inspect. Public, reproducible evidence provides stronger verification.

## Can I unlock achievements in my own repository?

Some achievements can be triggered through legitimate activity in repositories you control. The work should still be substantive, accurately attributed, and consistent with GitHub's rules.

## Why does the encyclopedia reject empty or spam pull requests?

The purpose of the project is to document responsible GitHub participation. Empty, deceptive, or repetitive activity damages repositories and produces weak evidence.

## What is the difference between verified and observed?

**Verified** means the behaviour has been reproduced with clear evidence. **Observed** means credible reports exist, but the project has not yet reproduced the result to the same standard.

## Can retired achievements still be earned?

Generally, retired achievements cannot be newly earned through current activity. Historical badges may remain visible on profiles that received them earlier.

## How can I submit evidence?

Use the [evidence submission guide](evidence-submission.md). Include the achievement, date, qualifying event, relevant URLs, account context, and any necessary privacy redactions.

## Is this project affiliated with GitHub?

No. The GitHub Achievement Encyclopedia is an independent community reference and is not an official GitHub publication.
